Image Dehazing using Dark Channel Prior (DCP)

📌 Overview

This project focuses on improving the visibility of images affected by fog, smoke, and haze using the Dark Channel Prior (DCP) algorithm.
The method enhances contrast, restores colors, and produces a clearer image.

---

🧠 Methodology (Simple)

- Read hazy input image
- Estimate dark channel
- Calculate atmospheric light
- Compute transmission map
- Recover clear image
- Apply contrast enhancement
